**题意：** 写一个函数，求两个整数之和，要求在函数体内不得使用 “+”、“-”、“*”、“/” 四则运算符号。

**题解：** 
* 使用异或运算符，我们可以将两个相同数位上只出现一次1的结果加起来
* 如果出现两次1，我们就用与操作记录并往上进位
* 重复这个过程直至没有进位。

```java
class Solution {
    public int add(int a, int b) {
        while(b != 0) {
            int c = (a & b) << 1;
            a = a ^ b;
            b = c;
        }
        return a;
    }
}
```
