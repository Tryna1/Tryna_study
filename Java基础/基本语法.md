#### 数据类型

在Java中，数据类型只有`四类八种`
* 整数型：byte、short、int、long
* 浮点型：float、double
* 字符型：char
* 布尔型：boolean


| 数据类型 | 内存 | 最小值 | 最大值 | 默认值 |
|--|--|--|--|--|
| byte | 8位 | -128 | 127 | 0 |
| short | 16位 | $-2^{16}$ | $2^{16}-1$ | 0 |
| int  | 32位 | $-2^{32}$ | $2^{32}-1$ | 0 |
| long | 64位 | $-2^{64}$ | $2^{64} - 1$ | 0L |
| float | 32位 | IEEE754 | IEEE754 | 0.0f |
| double | 64位 | IEEE754 | IEEE754 | 0.0d |
| char | 16位 | Unicode 0 | Unicode $2^{16} - 1$ | \u0000 |
| boolean | 8位 | - | - | false |
